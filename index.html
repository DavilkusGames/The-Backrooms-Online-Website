Multiplayer survival horror game developed by Davilkus Games LLC, based on the Backrooms fandom.

The game is an immersive exploration of multiple Backrooms levels with friends or via matchmaking. The project aims to create a deeply atmospheric and mechanically rich experience, bringing the unsettling lore of this specific Backrooms level to life.
